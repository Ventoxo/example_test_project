# Проект для тестирования сервиса "example_url.com"

### Стек

* Язык программирования: **Python**
* Основная библиотека: **requests**

### Структура проекта

```
 ├── data                                   # Папка для констант и статических тестовых данных
 │   ├── enum                               # Папка для хранения тестовых данных
 │   │   └── errors_enum.py                 # Файл для хранения ошибок сервиса
 │   │   └── hero_names_enum.               # Файл для хранения имен героев
 │   │   └── urls_enum.py                   # Файл для хранения контрактов сервиса
 ├── helpers                                # Папка для вспомогательного функционала общего назначения 
 │   └── get_random_name_of_character.py    # Файл для вспомогательных функций случайного выбора имени
 ├── logic                                  # Папка для хранения логики работы с функционалом сервиса
 │   └── client_logic                       # Папка с созданием клиента, а также базовые requests-методы
 ├── tests                                  # Папка для хранения тестов
 │   └── conftest.py                        # Файл для хранения фикстур
 └── README.md                              # Файл с описанием репозитория для тестов
```
